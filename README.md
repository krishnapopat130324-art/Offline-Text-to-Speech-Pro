# 🗣️ Offline Text-to-Speech Pro

### Powerful Offline Text-to-Speech Desktop Application

Offline Text-to-Speech Pro is a powerful desktop application that converts text into natural speech entirely offline. Built with Python and Tkinter, it uses the built-in Windows speech engine to deliver fast, reliable, and completely private text-to-speech functionality without requiring an internet connection.

The application is designed for students, professionals, language learners, visually impaired users, and anyone who wants an efficient offline reading companion.

---

## 📖 Overview

Unlike cloud-based text-to-speech services, Offline Text-to-Speech Pro operates completely locally on your machine.

### Key Benefits

* 100% Offline Processing
* Completely Free to Use
* No API Keys Required
* No Subscriptions
* No Data Collection
* Complete Privacy

## 🚀 Installation

### Prerequisites

* Python 3.6 or higher
* Windows Operating System

---

### Step 1: Clone Repository

```bash
git clone https://github.com/krishnapopat130324-art/Offline-Text-to-Speech-Pro.git
cd Offline-Text-to-Speech-Pro
```

### Step 2: Create Virtual Environment (Optional)

#### Windows

```bash
python -m venv venv
venv\Scripts\activate
```

#### Linux / macOS

```bash
python -m venv venv
source venv/bin/activate
```

### Step 3: Install Dependencies

```bash
pip install -r requirements.txt
```

### Step 4: Run Application

```bash
python main.py
```

---

## 🎯 How to Use

1. Enter or paste text into the input area.
2. Click **Play** to start speech synthesis.
3. Use **Pause** and **Stop** controls when required.
4. Select preferred voice settings.
5. Adjust speed and volume.
6. Import documents for reading.
7. Export speech as audio files.
